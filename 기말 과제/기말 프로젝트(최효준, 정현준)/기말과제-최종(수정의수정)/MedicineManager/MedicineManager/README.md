# MedicineManager (약품 유통기한 관리 프로그램)

----------------

## 1. 프로그램 소개
-  본 프로그램은 가정 내 상비약의 유통기한을 체계적으로 관리하여 의약품 오남용을 방지하기 위해 개발되었습니다. 
   C# WPF를 기반으로 제작되었으며, 직관적인 UI(달력, 차트)를 통해 사용자가 약품 정보를 손쉽게 등록하고 유통기한 임박 여부를 시각적으로 확인할 수 있는 개인 맞춤형 약품 관리 솔루션 프로그램입니다.

## 2. 사용 기술
- 언어 : C#

- 외부 라이브러리 : System.Data.SQLite (데이터베이스 연동)

- 프레임워크 : WPF (Windows Presentation Foundation), .NET (net9.0-windows)

- 데이터베이스 : SQLite

- 제작 프로그램 : Visual Studio 2022



## 3. 주요 기능
### 3.1 회원가입 및 로그인
- 회원가입 : 아이디 중복 검사, 비밀번호 확인, 전화번호 자동 하이픈(-) 처리 및 유효성 검사.
- 로그인 : SQLite DB 연동 인증, 로그인 성공 시 사용자 세션(CurrentUser) 유지.

### 3.2 약품 등록 및 관리 기능 (RegisterPage)
- 약품 등록 : 제품명, 종류(알약/물약 등), 유통기한, 메모, 사진 등록 기능.
- 목록 관리 : 등록된 약품 리스트 조회, 제품명 검색, 항목별(이름/종류/기한) 정렬 기능.
- 삭제 : 체크박스 선택을 통한 다중 삭제 기능 지원.

### 3.3 약품 상세 정보 수정 (DetailMedicineWindow)
- 상세 조회 : 리스트 더블 클릭 시 약품의 상세 정보(이미지 포함) 확인.
- 정보 수정 : 기등록된 약품의 정보를 수정하고 DB에 반영하는 기능.

### 3.4 달력 기반 유통기한 관리 (CalendarPage)
- 시각적 알림 : 유통기한 만료일이 포함된 날짜를 달력에 색상으로 강조 표시.
- 일별/월별 조회 : 특정 날짜 클릭 시 만료 약품 팝업 표시, 하단 리스트에 해당 월 만료 예정 약품 출력.

### 3.5 통계 분석 (ChartPage)
- 원형 차트 : 보유 중인 약품의 종류별(알약, 물약, 연고 등) 비율 시각화.
- 막대 차트 : 향후 6개월간 월별 폐기 예상 약품 수량을 막대그래프로 표시.

### 3.6 마이페이지 (MyPage)
- 정보 관리 : 사용자 프로필 이미지 변경 및 개인정보(이름, 전화번호 등) 수정, 로그아웃 기능.



## 4. 실행 방법
### 4.1 프로그램 실행
1. 'MedicineManager.sln' 파일을 Visual Studio 2022로 엽니다.
2. 솔루션 빌드 후 실행(F5)을 합니다.
3. 최초 실행 시 'bin/Debug' 폴더 내에 'medicine.db' 파일이 자동으로 생성됩니다.

### 4.2 프로그램 사용 흐름
1. 시작 화면 : 회원가입을 통해 계정을 생성하고 로그인합니다.
2. 메인 대시보드 (Home) : 달력과 차트를 통해 현재 약품 현황을 한눈에 요약하여 확인합니다.
3. 약품 등록 (등록 탭) : '추가' 버튼을 눌러 관리할 약품 정보를 입력하고 사진을 첨부합니다.
4. 유통기한 확인 (달력 탭) : 분홍색으로 표시된 날짜를 클릭하여 만료되는 약을 확인합니다.
5. 통계 확인 (차트 탭) : 어떤 종류의 약이 많은지, 언제 약을 버려야 하는지 통계로 확인합니다.
6. 마이페이지(MyPage) : 개인정보를 수정하거나 로그아웃 합니다.



## 5. 개선 해야 될 점
1. 자동 알림 : 알림 등을 통해 유통기한 임박 시 푸시 알림을 주는 기능 추가.
2. UI/UX 고도화 : 사용자 경험을 고려한 디자인 개선 및 애니메이션 효과 추가.
3. 데이터 백업 : 로컬 DB(SQLite) 외에 서버 연동이나 CSV 내보내기를 통한 데이터 백업/복원 기능 추가.
4. 관리자 모드 및 공지사항 기능 : 현재는 공지사항 버튼(UI)만 존재하지만, 추후 관리자 계정(Administrator) 기능을 도입하여 관리자가 공지사항을 작성하고 일반 사용자가 이를 조회할 수 있는 게시판 기능을 구현하고 싶습니다.
